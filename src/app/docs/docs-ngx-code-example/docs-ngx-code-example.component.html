@ngx-lite/code-example

The ngx-code-example component uses prismjs and Angular to easily display code examples in a view. 

      
<ngx-code-example>
  <![CDATA[
if (true) {{'{'}}
  console.log('hi');
{{'}'}} else {{'{'}}
  console.log('bye');
{{'}'}}
  ]]>
</ngx-code-example>
